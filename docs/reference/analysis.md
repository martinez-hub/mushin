# analysis

Offline, dependency-free views of a sweep directory. Each reads the per-cell
status and metrics sidecars directly (no Hydra or xarray), so they load instantly
and work mid-sweep. See the
[exploration → paper guide](../guides/exploration-to-paper.md) for how they fit an
exploration workflow.
